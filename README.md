>   - 저는 블로그, 노션, github 이외에도 제 노트북에 개발 학습 내용이 100개 이상 기록되어 있습니다.

>   - 개발을 하면서 숱한 에러를 마주할 것이고, 그 때마다 문제를 해결하려는 끈기가 중요하다고 생각합니다.
